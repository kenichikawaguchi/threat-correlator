# Threat Correlator

An open-source platform for collecting threat intelligence and correlating it with real-world attack observations.

## Vision

Threat Correlator aims to bridge the gap between public vulnerability intelligence and real-world attack activity.

## Roadmap

### v0.1
- NVD collector
- SQLite storage
- Daily markdown report

### v0.2

### v0.3
- Exploit-DB integration

### v0.4
- GitHub PoC monitoring
