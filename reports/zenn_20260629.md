# 2026年6月30日までの30日間で観測された26件の脆弱性（Critical13件・High3件・Unknown10件）

## はじめに
ハニーポット「AIBOT RADAR」は、インターネット上からの不正アクセス試行を受動的に収集することで、実際に攻撃者が狙っている脆弱性を把握します。今回のレポートは **2026年5月31日〜2026年6月29日の30日間** にハニーポットが検知した **26件** の CVE を対象に、観測回数・観測国・攻撃シナリオをまとめた実務向けの分析です。日本でも同様の攻撃が観測されているため、早急な対策が求められます。

## 🔴 Critical（CVSS 9.0 以上）の観測 CVE

### SAP SE SAP NetWeaver (Visual Composer development server)
- **CVE-2025-31324**  
  - **概要**：Visual Composer のメタデータアップローダが認証を行わずに実行可能バイナリのアップロードを許可している。攻撃者は任意のコードをサーバー上で実行でき、機密情報の漏洩やサービス停止が起こり得る。  
  - **攻撃シナリオ**：外部から HTTP POST で不正なバイナリを送信 → サーバ上で自動展開 → 任意コード実行 → 権限昇格・情報窃取。  
  - **推奨対策**：SAP の公式パッチ（2025‑31324）を即適用し、不要な Visual Composer ディベロップメントサーバーはネットワーク外に隔離。Web アプリケーションファイアウォール（WAF）で不審なファイルアップロードを遮断。  
  - **観測情報**：初回 2026‑05‑29、最終 2026‑06‑29、**92 回**、Netherlands・United States。

### Ivanti Ivanti Endpoint Manager Mobile
- **CVE-2023-35078**  
  - **概要**：認証バイパスにより、未認証のユーザーが管理 API に直接アクセスできる。結果として設定情報の取得や無許可の操作が可能になる。  
  - **攻撃シナリオ**：API エンドポイントへリクエスト送信 → 認証なしで管理機能実行 → デバイス情報漏洩・遠隔操作。  
  - **推奨対策**：Ivanti のセキュリティパッチを適用し、管理 API へのアクセスは VPN など内部ネットワークに限定。API キーのローテーションと監査ログの強化。  
  - **観測情報**：初回 2026‑06‑01、最終 2026‑06‑01、**1 回**、Netherlands。

### ConnectWise ConnectWise ScreenConnect
- **CVE-2024-1709**  
  - **概要**：認証バイパスが代替ルートで可能となり、攻撃者は画面共有セッションに不正に参加できる。機密情報の取得や遠隔操作が課題。  
  - **攻撃シナリオ**：特定の URL パラメータを改ざん → 認証を回避 → セッション取得 → デスクトップ閲覧・操作。  
  - **推奨対策**：ScreenConnect 23.9.7 以前のバージョンを最新（≥ 24.0）にアップデート。アクセス制御リスト（ACL）でリモート接続元を限定。  
  - **観測情報**：初回 2026‑06‑01、最終 2026‑06‑01、**1 回**、Netherlands。

### Roundcube Roundcube Webmail
- **CVE-2025-49113**  
  - **概要**：`_from` パラメータのバリデーション欠如により PHP オブジェクトデシリアライズが実行され、認証済みユーザーでリモートコード実行が可能になる。  
  - **攻撃シナリオ**：細工した URL を認証ユーザーにクリックさせる → デシリアライズ → 任意コード実行 → メールサーバー乗っ取り。  
  - **推奨対策**：Roundcube 1.5.10 以降にアップデート。Web サーバーの `open_basedir` 制限と WAF で不正なパラメータをブロック。  
  - **観測情報**：初回 2026‑06‑17、最終 2026‑06‑17、**1 回**、United Kingdom。

### Cisco Cisco HyperFlex HX Data Platform
- **CVE-2021-1497**  
  - **概要**：Web 管理インタフェースに複数のコマンドインジェクション脆弱性があり、未認証のリモート攻撃者がシステムコマンドを実行できる。  
  - **攻撃シナリオ**：特製リクエストを送信 → コマンドインジェクション → 任意シェル実行 → データ破壊・情報取得。  
  - **推奨対策**：Cisco のセキュリティアドバイザリで提供されたパッチを適用。管理インタフェースは IP アクセス制限と二要素認証で保護。  
  - **観測情報**：初回 2026‑06‑01、最終 2026‑06‑01、**1 回**、Netherlands。

### JetBrains TeamCity
- **CVE-2023-42793**（バージョン < 2023.05.4）  
- **CVE-2024-27198**（バージョン < 2023.11.4）  

  - **概要**：認証バイパスにより、未認証の攻撃者が管理者権限で操作できる。2024‑27198 は特に複数回観測され、欧州中心に攻撃が拡大。  
  - **攻撃シナリオ**：特定の API エンドポイントにリクエスト → 認証を回避 → ビルド設定変更・コード取得 → CI/CD パイプライン全体の乗っ取り。  
  - **推奨対策**：TeamCity を 2023.05.4 以上、できれば最新 LTS に更新。外部からの API アクセスは VPN 経由に限定し、ログ監視を強化。  
  - **観測情報**：  
    - CVE-2023-42793：初回 2026‑06‑01、最終 2026‑06‑01、**1 回**、Netherlands。  
    - CVE-2024-27198：初回 2026‑06‑01、最終 2026‑06‑01、**5 回**、Netherlands。

### LiteSpeed Technologies LiteSpeed Cache
- **CVE-2024-44000**  
  - **概要**：認証情報が十分に保護されておらず、認証バイパスが可能になる。攻撃者は WordPress 管理者権限を取得できる。  
  - **攻撃シナリオ**：プラグインの認証ページに直接リクエスト → バイパス成功 → 管理画面アクセス → プラグイン改ざん・マルウェア設置。  
  - **推奨対策**：LiteSpeed Cache 6.5.0.1 以上に更新し、`login_attempts` などの制限設定を有効化。  
  - **観測情報**：初回 2026‑06‑01、最終 2026‑06‑01、**1 回**、Netherlands。

### WP Automatic
- **CVE-2024-27954**  
  - **概要**：パス・トラバーサルと SSRF が可能で、任意ファイル取得や内部サーバへのリクエストが実行できる。  
  - **攻撃シナリオ**：プラグインのインストールパスに `/../../` を付与 → 任意ファイルダウンロード、または内部 API へリクエスト送信。  
  - **推奨対策**：WP Automatic 3.92.0 以上にアップデート。`open_basedir` の設定と、外部リクエストを制限するファイアウォールを導入。  
  - **観測情報**：初回 2026‑06‑01、最終 2026‑06‑01、**1 回**、Netherlands。

### Fortinet FortiOS / FortiProxy
- **CVE-2023-27997**  
  - **概要**：ヒープバッファオーバーフローにより、リモートコード実行が可能。特に SSL‑VPN のエンドポイントが標的になる。  
  - **攻撃シナリオ**：細工した SSL‑VPN パケット送信 → バッファオーバーフロー → 任意コード実行 → VPN トンネル取得・内部ネットワーク横滑り。  
  - **推奨対策**：FortiOS 7.2.4 以下、7.0.11 以下、6.4.12 以下、6.0.16 以下をすべて最新のパッチへ。不要な VPN 機能は無効化。  
  - **観測情報**：初回 2026‑06‑01、最終 2026‑06‑01、**2 回**、Netherlands。

### Progress MOVEit Transfer
- **CVE-2024-5806**  
  - **概要**：SFTP モジュールで認証バイパスが存在し、攻撃者が認証なしでファイル転送機能を利用できる。  
  - **攻撃シナリオ**：SFTP 接続を確立 → 認証を回避 → 任意ファイル取得・改ざん → データ漏洩。  
  - **推奨対策**：2023.0.11 以前、2023.1.6 以前、2024.0.2 以前のバージョンはすべてアップデート。SSH キー管理とアクセス制御リストを厳格化。  
  - **観測情報**：初回 2026‑06‑01、最終 2026‑06‑01、**1 回**、Netherlands。

### Fortinet FortiClientEMS
- **CVE-2026-21643**  
  - **概要**：SQL インジェクションにより、未認証の攻撃者が任意の SQL コマンドを実行できる。大量観測（**516 回**）は積極的な攻撃が進行中であることを示す。  
  - **攻撃シナリオ**：特製 HTTP GET に `UNION SELECT` などを埋め込んだリクエスト → データベース改ざん・認証情報取得 → 管理コンソール乗っ取り。  
  - **推奨対策**：FortiClientEMS 7.4.4 以前はすべて 7.4.5 以上に更新。Web アプリケーションファイアウォールで `UNION` 系クエリをブロックし、データベースの最小権限化を実施。  
  - **観測情報**：初回 2026‑06‑01、最終 2026‑06‑28、**516 回**、Australia・Belgium・Brazil・Chile・France・India・Indonesia・Japan・Netherlands・Singapore・South Korea・Sweden・Taiwan・United Kingdom・United States。

### Microsoft Exchange Server
- **CVE-2021-34473**（ProxyShell）  
  - **概要**：Exchange のフロントエンドプロキシに存在するリモートコード実行脆弱性。特権昇格後にサーバ全体が制御下に置かれる。  
  - **攻撃シナリオ**：特製 OWA リクエスト → 認証バイパス → PowerShell コマンド実行 → 邮件データ取得・スパム配信。  
  - **推奨対策**：Microsoft が提供する 2021‑12‑15 以降の累積パッチを適用。外部からの OWA アクセスは IP 制限で保護し、Exchange の管理者アカウントは MFA を必須化。  
  - **観測情報**：初回 2026‑06‑02、最終 2026‑06‑28、**13 回**、United States。

**ポイント**  
- **CVE‑2026‑21643** は 516 回の観測があり、国内外で広範に試行されているため最優先。  
- **CVE‑2025‑31324** も 92 回の観測と 10.0 のスコアで深刻。SAP 環境が社内にある場合は直ちにパッチを。  
- 高頻度で観測された **Grafana** の XSS は内部ユーザーの被害拡大を防ぐため、匿名アクセスを無効化し CSP を導入してリスクを低減。  

これらの対策を速やかに実施し、ハニーポットが示すリアルタイムの攻撃動向を踏まえて防御体制を強化しましょう。
