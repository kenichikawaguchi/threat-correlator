# 📊 2026年5月末〜6月末（直近30日）に観測されたサイバー脅威 — 全26件（Critical 13件・High 3件・CVSS不明 10件）

## はじめに
ハニーポット「AIBOT RADAR」は、外部からの不正アクセスやマルウェア感染を模倣したサービスをインターネット上に設置し、実際に試みられた攻撃手法を収集・分析するプラットフォームです。今回、直近30日間に同ハニーポット（日本向けエンドポイント含む）で観測された **26件** の脆弱性情報を集計しました。  
- **観測対象**：Webアプリケーション、サーバー管理コンソール、メールシステム等、企業で広く利用される製品群  
- **観測方法**：不正アクセス試行を自動で記録し、ET（Emerging Threats）シグネチャと結び付けて脆弱性ごとにカウント  
- **目的**：日本のセキュリティエンジニア・システム管理者が「今すぐ対応すべき」脆弱性を把握し、優先的にパッチ適用や設定変更を行えるようにすること  

以下に、Critical（CVSS 9.0 以上）を中心に詳細を解説します。

---

## 🔴 Critical（CVSS 9.0 以上）の観測 CVE

### SAP NetWeaver (Visual Composer development server)

- **CVE-2025-31324**  
  - **概要**：Visual Composer のメタデータアップローダが認証を行わずにバイナリを受け付けるため、任意の実行可能ファイルをサーバーにアップロードできる。  
  - **攻撃シナリオ**：攻撃者は `/visualcomposer/upload` エンドポイントへ不正バイナリを送信し、サーバ上で権限昇格または情報窃取を実行。  
  - **推奨対策**：最新パッチ（SAP Note 3594142）を適用し、アップロード先ディレクトリの実行権限を削除。外部からの直接アクセスをファイアウォールで制限。  
  - **観測情報**：2026‑05‑29 に初回観測、2026‑06‑29 に最終観測、**92 回** の試行、観測国は **Local, Netherlands, United States**。

### Ivanti Endpoint Manager Mobile

- **CVE-2023-35078**  
  - **概要**：認証バイパスにより、未認証の攻撃者が管理APIへ直接アクセスできる。  
  - **攻撃シナリオ**：APIエンドポイント `/api/v1/device` に認証情報なしでリクエストを送信し、デバイス情報取得や設定変更を実行。  
  - **推奨対策**：Ivanti が提供する 2023‑12‑01 以降のパッチを即適用。不要な外部公開APIは無効化し、IP 制限を導入。  
  - **観測情報**：2026‑06‑01 に初回・最終観測、**1 回** の試行、観測国は **Netherlands**。

### ConnectWise ScreenConnect

- **CVE-2024-1709**  
  - **概要**：認証バイパスが別経路で可能となり、攻撃者が管理画面へ直接アクセスできる。  
  - **攻撃シナリオ**：`/auth/alternate` 経由で認証情報なしにログインし、遠隔デスクトップ機能を悪用。  
  - **推奨対策**：23.9.7 以前のバージョンは 23.9.8 以降へアップデート。  
  - **観測情報**：2026‑06‑01 に単発観測、**1 回**、観測国 **Netherlands**。

### Roundcube Webmail

- **CVE-2025-49113**  
  - **概要**：`_from` パラメータの検証不足により PHP オブジェクトデシリアライズが可能。認証ユーザーが任意コード実行できる。  
  - **攻撃シナリオ**：悪意ある URL をメールに埋め込み、管理者がクリックすると `/actions/settings/upload.php` が不正オブジェクトをデシリアライズ。  
  - **推奨対策**：1.5.10 以上へのアップデート、_from パラメータのホワイトリスト化。  
  - **観測情報**：2026‑06‑17 に単発観測、**1 回**、観測国 **United Kingdom**。

### Cisco HyperFlex HX Data Platform

- **CVE-2021-1497**  
  - **概要**：管理コンソールのコマンドインジェクション脆弱性。認証不要でシェルコマンド実行が可能。  
  - **攻撃シナリオ**：`/admin/execute` に特製パラメータを送付し、システムコマンド `rm -rf /` を実行。  
  - **推奨対策**：Cisco が提供する最新セキュリティアドバイザリに基づくパッチ適用。管理インタフェースへの IP 制限。  
  - **観測情報**：2026‑06‑01 に単発観測、**1 回**、観測国 **Netherlands**。

### JetBrains TeamCity

- **CVE-2023-42793**（< 2023.05.4）  
- **CVE-2024-27198**（< 2023.11.4）  

  - **概要**：両脆弱性とも認証バイパスにより、攻撃者が管理者権限で任意のビルドや設定変更を行える。  
  - **攻撃シナリオ**：`/teamcity/httpAuth/app/rest` に不正リクエストを投げ、ビルドスクリプトを改竄。  
  - **推奨対策**：2023.05.4 以上あるいは 2023.11.4 以上へアップデート。外部からの API アクセスは VPN のみ許可。  
  - **観測情報**：両 CVE 2026‑06‑01 に初回・最終観測。CVE-2023-42793 は **1 回**、CVE-2024-27198 は **5 回**、観測国は **Netherlands**。

### LiteSpeed Cache

- **CVE-2024-44000**  
  - **概要**：認証情報が不適切に保護され、認証バイパスが可能。攻撃者は管理者アカウントを乗っ取れる。  
  - **攻撃シナリオ**：`/wp-admin/admin-ajax.php` 経由で認証トークン取得後、管理画面へ不正ログイン。  
  - **推奨対策**：6.5.0.1 以上にアップデート、管理画面への二要素認証導入。  
  - **観測情報**：2026‑06‑01 に単発観測、**1 回**、観測国 **Netherlands**。

### WP Automatic

- **CVE-2024-27954**  
  - **概要**：パス・トラバーサルと SSRF が同時に可能。任意ファイル取得や内部サーバへのリクエストが実行できる。  
  - **攻撃シナリオ**：`/wp-content/plugins/automatic/` に `../../../../etc/passwd` を指定しファイル取得、続けて内部 API へリクエスト送信。  
  - **推奨対策**：3.92.0 以上へアップデート、プラグインのディレクトリ書き込み権限を最小化。  
  - **観測情報**：2026‑06‑01 に単発観測、**1 回**、観測国 **Netherlands**。

### Fortinet FortiOS / FortiProxy

- **CVE-2023-27997**  
  - **概要**：ヒープベースのバッファオーバーフローにより、特定の SSL‑VPN リクエストでコード実行が可能。  
  - **攻撃シナリオ**：カスタム TLS ハンドシェイクを送信し、メモリ上の制御構造体を書き換えてシェルコード実行。  
  - **推奨対策**：7.2.4 未満、7.0.11 未満、6.4.12 未満、6.0.16 未満のバージョンはすべてアップデート。SSL‑VPN の外部公開は非推奨。  
  - **観測情報**：2026‑06‑01 に単発観測、**2 回**、観測国 **Netherlands**。

### Progress MOVEit Transfer

- **CVE-2024-5806**  
  - **概要**：SFTP モジュールの認証バイパスにより、攻撃者が不正にログインできる。  
  - **攻撃シナリオ**：`/sftp` エンドポイントに不正なクレデンシャルを送信し、ファイル転送権限を取得。  
  - **推奨対策**：2023.0.11、2023.1.6、2024.0.2 未満からのアップデート。SFTP の IP 制限を導入。  
  - **観測情報**：2026‑06‑01 に単発観測、**1 回**、観測国 **Netherlands**。

### Fortinet FortiClientEMS

- **CVE-2026-21643**  
  - **概要**：SQL インジェクションにより、認証なしで任意の SQL が実行可能。  
  - **攻撃シナリオ**：`/api/v1/agents?filter=id=1 OR 1=1` で管理者情報取得、続けてコマンド実行。  
  - **推奨対策**：7.4.4 以上へアップデート、Web アプリケーションファイアウォール (WAF) でパラメータ検証を強化。  
  - **観測情報**：2026‑06‑01 に初回、2026‑06‑28 に最終、**516 回** の大量試行、観測国は **Australia, Belgium, Brazil, Chile, France, India, Indonesia, Japan, Netherlands, Singapore, South Korea, Sweden, Taiwan, United Kingdom, United States**。

### Microsoft Exchange Server

- **CVE-2021-34473**（ProxyShell）  
  - **概要**：リモートコード実行（RCE）に至る複合的な認証バイパスと権限昇格。  
  - **攻撃シナリオ**：Exchange の `/ecp` エンドポイントに特製リクエストを送信し、`powershell.exe` をリモートで起動。  
  - **推奨対策**：該当累積アップデート（CU23, CU9 等）を適用し、ECP の外部公開を停止、メールサーバへの IP アクセス制御。  
  - **観測情報**：2026‑06‑02 に初回、2026‑06‑28 に最終、**13 回**、観測国 **Local, United States**。
