# 2026年5月30日〜2026年6月29日のハニーポット観測結果（計26件）

## はじめに
ハニーポット（AIBOT RADAR）は、インターネット上に意図的に脆弱性を持つサービスを配置し、実際の攻撃者の活動を観測・記録する仕組みです。今回のレポートは、直近30日間にハニーポットで検出された **26件** の CVE を対象に、観測回数・攻撃源国・攻撃手法を可視化したものです。データはハニーポットが受信したリクエストを自動的に解析し、CVE のメタデータと突合させることで取得しました。実際に日本国内からもプローブが報告されており、早急な対策が求められます。

## 🔴 Critical（CVSS 9.0 以上）の観測 CVE

以下は **Critical** ラベルが付与された 13 件の脆弱性です。各項目には *初回観測日・最終観測日・観測回数・観測国* を明記し、攻撃シナリオと推奨対策を併記しています。

### SAP SE SAP NetWeaver (Visual Composer development server) – CVE‑2025‑31324  
- **観測情報**：2026‑05‑29 （初回）〜 2026‑06‑29 （最終）・**92回**・Local, Netherlands, United States  
- **概要**：Visual Composer のメタデータアップローダーが認証チェックを行わないため、認証されていない攻撃者が任意の実行可能バイナリをサーバーにアップロードできる。結果として機密性・完全性・可用性が深刻に侵害される可能性がある。  
- **攻撃シナリオ**  
  1. 攻撃者が `/vc/metadata/upload` などのエンドポイントへ直接 POST リクエストを送信。  
  2. 任意のマルウェアバイナリを添付し、サーバー上に展開。  
  3. 以降、サーバー上でコードが実行され、データ窃取や横展開が可能になる。  
- **推奨対策**  
  - 該当コンポーネントを最新パッチ（SAP Note 3594142）に更新。  
  - アップロード機能をファイアウォールで外部からのアクセスを遮断。  
  - アップロードされたファイルの実行権限を厳格に制御し、ウイルススキャンを実施。

### Ivanti Endpoint Manager Mobile – CVE‑2023‑35078  
- **観測情報**：2026‑06‑01 （初回・最終）・**1回**・Netherlands  
- **概要**：認証バイパスにより、未認証の攻撃者が管理画面の機能やデータにアクセスできる。  
- **攻撃シナリオ**  
  1. 特定の API エンドポイントへ認証ヘッダーなしでリクエスト。  
  2. サーバー側で認証チェックが抜けているため、管理機能が利用可能になる。  
- **推奨対策**  
  - Ivanti の 2023‑06 パッチを適用。  
  - 不要な API エンドポイントは無効化し、外部からのアクセスは VPN に限定。  

### ConnectWise ScreenConnect – CVE‑2024‑1709  
- **観測情報**：2026‑06‑01 （初回・最終）・**1回**・Netherlands  
- **概要**：認証バイパスが別経路で可能となり、攻撃者が遠隔から機密情報やシステムへ直接アクセスできる。  
- **攻撃シナリオ**  
  1. 正規のログインフローを迂回し、リモートデスクトップセッションを開始。  
  2. 管理者権限で操作可能になる。  
- **推奨対策**  
  - 23.9.7 以前のバージョンは即時に 24.0 へアップグレード。  
  - 多要素認証 (MFA) を導入し、IP アクセス制限を実装。  

### Roundcube Webmail – CVE‑2025‑49113  
- **観測情報**：2026‑06‑17 （初回・最終）・**1回**・United Kingdom  
- **概要**：`_from` パラメータの検証不備により、PHP オブジェクトデシリアライズが可能となり、認証ユーザーがリモートコード実行できる。  
- **攻撃シナリオ**  
  1. 認証済みユーザーが細工した URL（`upload.php?_from=…`）へアクセス。  
  2. 悪意あるオブジェクトがサーバー側でデシリアライズされ、任意コードが実行。  
- **推奨対策**  
  - Roundcube 1.5.10 以上へアップデート。  
  - Web アプリケーションファイアウォール (WAF) でオブジェクトデシリアライズ系リクエストをブロック。  

### Cisco HyperFlex HX Data Platform – CVE‑2021‑1497  
- **観測情報**：2026‑06‑01 （初回・最終）・**1回**・Netherlands  
- **概要**：Web 管理インターフェイスに複数の脆弱性があり、認証なしでコマンドインジェクションが可能。  
- **攻撃シナリオ**  
  1. 特定の管理 API に細工したパラメータを送信。  
  2. OS シェルコマンドが実行され、システム権限取得。  
- **推奨対策**  
  - Cisco が提供する 2022 リリース以降のパッチを適用。  
  - 管理インターフェイスは社内ネットワークまたは VPN のみからアクセス可能に限定。  

### JetBrains TeamCity (バージョン < 2023.05.4) – CVE‑2023‑42793  
- **観測情報**：2026‑06‑01 （初回・最終）・**1回**・Netherlands  
- **概要**：認証バイパスにより、未認証のユーザーが RCE を実行できる。  
- **攻撃シナリオ**  
  1. 認証が不要なエンドポイントへリクエスト。  
  2. 任意のビルドステップがサーバー上で実行され、コードが走る。  
- **推奨対策**  
  - 2023.05.4 以上へアップデート。  
  - アクセス制御を見直し、未認証の API を無効化。  

### JetBrains TeamCity (バージョン < 2023.11.4) – CVE‑2024‑27198  
- **観測情報**：2026‑06‑01 （初回・最終）・**5回**・Netherlands  
- **概要**：認証バイパスにより管理者権限操作が可能になる。  
- **攻撃シナリオ**  
  1. 認証なしで管理者向け API にアクセス。  
  2. プラグインインストールや設定変更を実行。  
- **推奨対策**  
  - 2023.11.4 以降にアップデート。  
  - 管理 API へは IP ホワイトリストで制限。  

### LiteSpeed Cache – CVE‑2024‑44000  
- **観測情報**：2026‑06‑01 （初回・最終）・**1回**・Netherlands  
- **概要**：認証情報が適切に保護されておらず、認証バイパスが可能。  
- **攻撃シナリオ**  
  1. `/wp-admin/admin-ajax.php?action=ls_cache` などのエンドポイントに直接アクセス。  
  2. 管理者権限でキャッシュ設定が変更され、任意コード実行につながる。  
- **推奨対策**  
  - LiteSpeed Cache 6.5.0.1 以上へ更新。  
  - 管理画面へのアクセスは基本認証または MFA を併用。  

### WP Automatic – CVE‑2024‑27954  
- **観測情報**：2026‑06‑01 （初回・最終）・**1回**・Netherlands  
- **概要**：パス・トラバーサルと SSRF が組み合わさり、任意ファイル取得や内部リクエストが実行可能。  
- **攻撃シナリオ**  
  1. `/wp-content/plugins/automatic/.../../../../../etc/passwd` のようにパスを細工。  
  2. サーバー内部へのリクエストが走り、機密情報が漏洩。  
- **推奨対策**  
  - バージョン 3.92.0 以上へアップデート。  
  - プラグインフォルダへの書き込み権限を最小化し、WAF でパス・トラバーサルを検知。  

### Fortinet FortiOS / FortiProxy – CVE‑2023‑27997  
- **観測情報**：2026‑06‑01 （初回・最終）・**2回**・Netherlands  
- **概要**：ヒープバッファオーバーフローにより、リモートから任意コード実行が可能。VPN 機能が狙い目。  
- **攻撃シナリオ**  
  1. 脆弱な SSL‑VPN エンドポイントへ特製リクエスト送信。  
  2. メモリ破壊 → 任意コード実行 → ファイアウォール取得。  
- **推奨対策**  
  - FortiOS 7.2.5 以降、FortiProxy 7.2.4 以降へアップデート。  
  - VPN アクセスは MFA と IP 制限で保護。  

### Progress MOVEit Transfer – CVE‑2024‑5806  
- **観測情報**：2026‑06‑01 （初回・最終）・**1回**・Netherlands  
- **概要**：SFTP モジュールの認証バイパスにより、認証なしでファイル転送機能が利用可能になる。  
- **攻撃シナリオ**  
  1. `/api/sftp/login` へ不正リクエスト送信。  
  2. 認証が免除され、任意ファイルのアップロード/ダウンロードが可能。  
- **推奨対策**  
  - 2023.1.6 以上、2024.0.2 以上へアップデート。  
  - SFTP アクセスは内部ネットワークのみに限定し、ログ監視を強化。  

### Fortinet FortiClientEMS – CVE‑2026‑21643  
- **観測情報**：2026‑06‑01 （初回）〜 2026‑06‑28 （最終）・**516回**・Australia, Belgium, Brazil, Chile, France, India, Indonesia, **Japan**, Netherlands, Singapore, South Korea, Sweden, Taiwan, United Kingdom, United States  
- **概要**：SQL インジェクションにより、認証されていない攻撃者が任意の SQL を実行でき、システム全体の制御が奪われる可能性がある。  
- **攻撃シナリオ**（特に注目）  
  1. `/api/v1/users?filter=...` のクエリパラメータに `' OR '1'='1` などを挿入。  
  2. データベースに不正な SELECT/INSERT が実行され、管理者権限の情報取得やバックドア設置に至る。  
  3. **観測回数が 500 回超** であることから、世界規模で自動化されたスキャンが行われていることが判明。日本国内でも多数のプローブが確認されている。  
- **推奨対策**  
  - 7.4.4 以前のバージョンは直ちに 7.5.0 以降へアップデート。  
  - データベース接続はパラメータ化クエリに切り替え、Web アプリケーションファイアウォールで SQLi パターンをブロック。  
  - 不要なインターネット露出は遮断し、管理 UI へのアクセスは VPN + MFA に限定。  

### Microsoft Exchange Server – CVE‑2021‑34473  
- **観測情報**：2026‑06‑02 （初回）〜 2026‑06‑28 （最終）・**13回**・Local, United States  
- **概要**：所謂「ProxyShell」脆弱性で、認証なしにリモートコード実行が可能。  
- **攻撃シナリオ**  
  1. `/ecp/` への認証バイパスリクエストを送信。  
  2. サーバー側で PowerShell コマンドが実行され、権限昇格・データ窃取が行われる。  
- **推奨対策**  
  - Microsoft が提供する 2021 年 7 月以降の累積的パッチを即適用。  
  - **外部からの ECP アクセスは全て遮断**し、内部からは MFA を必須化。

## まとめと対策の優先順位

1. **緊急更新が必要な脆弱性（観測回数が多数）**  
   - **CVE‑2026‑21643（FortiClientEMS）**：516 件の観測。即座に最新バージョンへアップデートし、WAF で SQLi パターンをブロック。  
   - **CVE‑2025‑31324（SAP NetWeaver）**：92 件の観測。パッチ適用とアップロード機能の外部遮断を実施。  

2. **組織内で広く利用されている SaaS／メール系**  
   - **CVE‑2021‑34473（Microsoft Exchange）**：13 件。累積パッチ適用と外部 ECP アクセス遮断。  
   - **CVE‑2025‑49113（Roundcube）**：1 件だがメール環境での影響が大きい。バージョン 1.5.10 以上へ更新。  

3. **認証バイパス系は多数のベンダーが対象**  
   - **JetBrains TeamCity（CVE‑2023‑42793, CVE‑2024‑27198）**：認証なしで管理操作が可能。最新版へ更新し、IP ホワイトリストを適用。  
   - **Ivanti EPMM、ConnectWise ScreenConnect、LiteSpeed Cache**：それぞれ 1 件ずつだが、認証バイパスは遠隔からのフル侵入につながるため即時対策。  

4. **High スコアの XSS・情報漏洩系**  
   - **Grafana（CVE‑2025‑4123）**：268 回観測。匿名アクセス無効化とパッチ適用。  

5. **その他の中リスク CVE**（CVSS 不明や観測回数が少ないもの）  
   - 上記テーブルの通り、**パッチ適用、WAF ルール追加、不要サービス遮断** を段階的に実施。特に **CVE‑2022‑22274（SonicOS）** は 16 回観測されているので、ファイアウォールのファームウェア更新を優先。
